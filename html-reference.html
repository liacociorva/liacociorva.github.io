HTML Element

Description

The HTML and DOCTYPE tags define the laguage of the document. These are placed at the beginning of the document, and the </html> tag is placed at the very end. The slash sign is used in a pair of tags to indicate the start and end of the tag.

Syntax

Start tag:<html>, End tag:</html> Document tag: <!DOCTYPE html>

Example

<html></html>

Head Element

Description

The head tag is used to define information about the document that will not be displayed on the actual page. This can inlude titles, styles, and more. It is placed at the top of the page so that it is what the computer reads over first.

Syntax

Start tag:<head>, end tag:</head>

Example

<head>This is the header.</head>

Title Element

Description

The title attribute is used to title the document and it is what appears in the title of a tab when you open it. This is placed in the header.

Syntax

Start tag:<title>, end tag:</title>

Example

<title>This is a title.</title>

Body Element

Description

The body attriute is what HTML uses to know where the content of the site is actually going to be displayed. This is where content like text, images, and more are placed to appear on the page. It is usually placed somehwere in the middle of the document, but can go anywhere between the <html>/</html> tags.

Syntax

Start tag:<body>, end tag:<body>

Example

<body><h1>This is a heading.</h1></body>

Description

Syntax

Example
